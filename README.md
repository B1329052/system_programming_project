# Small-C 互動式直譯器

本系統為 Small-C 語言的互動式直譯器（Interpreter），以 Python 語言實作，作為系統程式課程之專題成果。系統提供一互動式命令列環境（REPL），支援 Small-C 程式之輸入、編輯、語法檢查與直譯執行。

## 系統需求與環境

* **執行環境**：Python 3.10 或更高版本。
* **依賴套件**：僅使用 Python 標準函式庫，無任何外部依賴。
* **支援平台**：相容於 Windows、macOS 與 Linux 作業系統。

## 啟動與執行

於終端機執行下列命令啟動直譯器：

```bash
python main.py
```

啟動後，系統進入互動式命令列介面，提示符號為 `sc>`。

## 系統結構

本系統由下列模組構成：

* `main.py`：主程式入口，負責互動式介面（REPL）之命令解析與排程。
* `lexer.py`：詞法分析器（Lexical Analyzer），將輸入之原始碼拆解為記號（Tokens）。
* `parser.py`：語法分析器（Parser），依據文法規則將記號串列建構為抽象語法樹（AST）。
* `interpreter.py`：直譯執行器，藉由走訪抽象語法樹執行對應之語義運作。
* `environment.py`：環境管理器，負責變數、函式、作用域與記憶體空間之模擬。
* `smallc_builtins.py`：內建函式庫，提供輸入輸出、字串處理與數學運算等基本常式。
* `README.md`：系統說明文件。

## 執行流程

直譯器處理原始碼之流程如下：

```
原始碼 ──> 詞法分析 (Lexer) ──> 記號串列 ──> 語法分析 (Parser) ──> 抽象語法樹 (AST) ──> 直譯執行 (Interpreter) ──> 輸出
```

1. **詞法分析**：將原始碼字元流解析為具語義之記號（如識別字、常數、運算子）。
2. **語法分析**：依文法規則將記號串列轉換為以 Python `dict` 表示之抽象語法樹（AST）。
3. **直譯執行**：遞迴走訪 AST 節點，依語義執行運算、變數存取或函式呼叫。
4. **環境管理**：維護變數與函式之符號表，模擬全域、函式及區塊等巢狀作用域。

## Small-C 語言規格

本系統支援之 Small-C 語法子集定義如下：

### 資料型別

* **基本型別**：
  * `int`：整數。
  * `char`：字元。
* **衍生型別**：
  * 陣列：如 `int arr[10]`（整數陣列）、`char s[50]`（字元陣列）。
  * 指標：如 `int *p`（整數指標）。

### 運算子與優先順序

下表依結合性與優先權由高至低列出支援之運算子：

* 括號：`()`
* 一元運算子：`+`, `-`, `!`, `&`（取位址）, `*`（解參照）
* 乘除與取餘數：`*`, `/`, `%`
* 加與減：`+`, `-`
* 移位：`<<`, `>>`
* 關係運算子：`<`, `<=`, `>`, `>=`
* 等值比較：`==`, `!=`
* 位元運算：`&`, `^`, `|`
* 邏輯運算：`&&`, `||`
* 指定運算子：`=`, `+=`, `-=`, `*=`, `/=`, `%=`

### 控制流程

* 條件分支：`if` / `else`
* 迴圈結構：`while`, `for`, `do ... while`
* 流程控制：`break`, `continue`, `return`

### 其他特性

* 支援函式定義與呼叫，支援遞迴（Recursion）。
* 註解格式：單行註解 `//` 與區塊註解 `/* ... */`。
* 前處理指令：`#define` 常數替換。

## 互動命令參考

直譯器環境支援下列控制指令（不區分大小寫）：

| 指令 | 語法與說明 |
| :--- | :--- |
| `ABOUT` | 顯示系統版本與專案資訊。 |
| `HELP` | 顯示指令使用說明。 |
| `APPEND` | 進入多行原始碼輸入模式。 |
| `LIST [n\|n1-n2]` | 列出緩衝區內之程式碼。可指定單行 `n` 或範圍 `n1-n2`。 |
| `EDIT n` | 編輯第 `n` 行程式碼。 |
| `DELETE n` | 刪除第 `n` 行程式碼。 |
| `INSERT n` | 於第 `n` 行前插入程式碼。 |
| `CHECK` | 進行語法檢查並建構抽象語法樹。 |
| `RUN` | 執行程式中之 `main()` 函式。 |
| `SAVE filename` | 將當前緩衝區之程式碼寫入指定檔案。 |
| `LOAD filename` | 自指定檔案載入程式碼至緩衝區。 |
| `NEW` | 清除當前程式碼與執行環境狀態。 |
| `TRACE ON/OFF` | 啟動或關閉執行期追蹤模式。 |
| `VARS` | 列出當前環境中已定義之變數及其數值。 |
| `FUNCS` | 列出當前環境中已定義之函式簽章。 |
| `CLEAR` | 清除終端機畫面。 |
| `QUIT` / `EXIT` | 結束直譯器程式。 |

## 內建函式庫

系統提供下列預先定義之函式以供程式呼叫：

### 輸入與輸出常式 (Input/Output)

* `printf(format, ...)`：格式化輸出，支援 `%d`, `%c`, `%s`, `%x` 格式說明符。
* `getchar()`：自標準輸入讀取單一字元。
* `putchar(ch)`：輸出單一字元至標準輸出。
* `puts(str)`：輸出字串並附加換行字元。
* `scanf(format, &var)`：格式化輸入，支援基本變數寫入。

### 字串處理常式 (String Handling)

* `strlen(str)`：計算字串長度。
* `strcpy(dest, src)`：複製字串。
* `strcat(dest, src)`：連接字串。
* `strcmp(s1, s2)`：比較兩字串之語意大小。

### 數學運算常式 (Mathematics)

* `abs(x)`：求絕對值。
* `max(a, b)` / `min(a, b)`：求最大值與最小值。
* `pow(base, exp)`：求冪次。
* `sqrt(x)`：求平方根。

### 公用程式 (Utilities)

* `atoi(str)`：將字串轉換為整數。
* `itoa(num, str)`：將整數轉換為字串。
* `rand()`：產生偽隨機數。
* `srand(seed)`：設定隨機數種子。
* `memset(ptr, value, size)`：充填特定記憶體區域。
* `sizeof(x)`：取得變數或型別之位元組大小。
* `isdigit(ch)` / `isalpha(ch)`：判斷字元是否為數字或英文字母。

## 測試與驗證

使用者可於直譯器啟動後，在 `sc>` 提示符後輸入單行敘述進行測試：

```
sc> printf("%d\n", 3 + 4 * 5 - 2);
21
sc> printf("%d\n", (3 + 4) * (5 - 2));
21
```

若欲執行完整程式，應依序執行下列步驟：

1. 鍵入 `APPEND` 命令進入程式碼輸入模式。
2. 輸入 Small-C 原始碼，結束後鍵入空行返回。
3. 鍵入 `CHECK` 檢查語法。
4. 鍵入 `RUN` 執行 `main` 函式。
